# ガーデンパズル 仕様書

## 概要
- マッチ3パズルでステージをクリアして庭を復興するゲーム
- フルーツ・花・宝石などを揃えて消す
- クリア条件を達成すると庭が少しずつ完成していく

## ゲームフロー
1. タイトル画面
2. ステージ選択
3. パズルプレイ
4. クリア演出
5. ガーデン復興演出
6. 次のステージへ

## パズルルール
- 同じアイテムを3つ以上並べると消える
- 連鎖でコンボ発生
- 目標回数以内にミッションを達成する

## ステージ目標例
- 赤い花を20個消す
- 木箱を5個破壊する
- スコア5000点以上獲得

## ガーデン要素
- クリアごとに庭のオブジェクトが解放される
- ベンチ・噴水・花壇などを配置

## UI
- スコア
- 残り手数
- 目標表示
- ガーデン進捗

## 技術
- Three.js または Canvas
- PC・スマホ対応
- GitHub Pages公開

## 実装指示（AI向け）

- まず Phase1 のみ実装する
- Three.js または Canvas でOK
- UIはシンプルでよい
- デザインより動作優先
